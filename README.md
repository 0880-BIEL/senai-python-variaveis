# 🎓 Aprendendo Variáveis em Python - SENAI

Repositório criado para salvar os exercícios e códigos da aula de Python no **SENAI**.

## 💻 O que foi visto na aula:
* **Variáveis:** Tipos primitivos (`int`, `float`, `string` e `complex`).
* **Números Complexos:** Cálculos e uso da biblioteca `cmath`.
* **Strings:** Fatiamento (`slicing`), caracteres de escape e métodos de manipulação (como `.upper()`, `.lower()`, `.split()`, `.join()`).
* **Validações:** Uso de funções como `isdigit()`, `isalpha()` e verificação de palavras reservadas com `iskeyword`.
